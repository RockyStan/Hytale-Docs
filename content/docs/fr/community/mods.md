## Types de Mods

### Categories

1. **Construction/Decoration** - Blocs decoratifs, meubles, fenetres, portes, ponts
2. **Qualite de Vie (QoL)** - Gestion d'inventaire, stockage ameliore, automatisation
3. **Gameplay** - Nouveaux modes de jeu (Skyblock, etc.), agriculture avancee, systemes musicaux
4. **Serveur/Multijoueur** - Protection de terrain, economies, mini-jeux
5. **Visuels** - Texture Packs (16x a 512x), Shaders (dont Ray Tracing), Packs PBR

### Types Techniques

| Type | Description | Prerequis |
|------|-------------|-----------|
| **Packs** | Packs d'assets/contenus pour nouveaux blocs, mobs, items | Aucun code - Asset Editor |
| **Plugins** | Mods Java utilisant l'API du jeu | Programmation Java |
| **Bootstrap Plugins** | Plugins bas niveau pour transformation de bytecode | Java avance |

## Informations Cles

### Architecture Server-Side
- **Tous les mods s'executent cote serveur**
- Aucune installation client requise
- Les joueurs recoivent automatiquement le contenu en se connectant
- Le serveur n'est pas obfusque (decompilation possible)

### Partenariat CurseForge (5 janvier 2026)
- Integration directe avec le client Hytale
- Installation en un clic
- Moderation proactive et scan de securite
- Statistiques d'adoption en temps reel pour les createurs
