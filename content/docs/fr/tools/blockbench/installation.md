# Installation de Blockbench

Blockbench est l'outil officiel pour creer des modeles 3D et des animations pour Hytale. Il remplace l'outil interne "Hytale Model Maker" pour mieux supporter les workflows creatifs etablis dans la communaute.

> "Nous supportons officiellement Blockbench pour creer des modeles, textures et animations Hytale."
> -- [Strategie de modding Hytale](https://hytale.com/news/2025/11/hytale-modding-strategy-and-status)

## Telecharger Blockbench

Obtenez Blockbench depuis le site officiel : [blockbench.net](https://www.blockbench.net/)

**Plateformes disponibles :**
- Windows
- macOS
- Linux
- Version web (fonctionnalites limitees)

## Installer le plugin Hytale

Le plugin officiel Hytale ajoute le support des formats et fonctionnalites specifiques a Hytale.

### Depuis le gestionnaire de plugins

1. Ouvrir Blockbench
2. Aller dans **File > Plugins**
3. Cliquer sur l'onglet **Available**
4. Rechercher "Hytale"
5. Cliquer sur **Install**

### Depuis GitHub

Alternativement, telecharger directement depuis le depot officiel :

**Depot GitHub :** [https://github.com/JannisX11/hytale-blockbench-plugin](https://github.com/JannisX11/hytale-blockbench-plugin)

1. Telecharger le fichier du plugin
2. Dans Blockbench, aller dans **File > Plugins**
3. Cliquer sur le bouton **Load Plugin from File**
4. Selectionner le fichier du plugin telecharge

## Verifier l'installation

Pour confirmer que le plugin est correctement installe :

1. Ouvrir Blockbench
2. Aller dans **File > New**
3. Chercher **Hytale Model** dans les types de projet
4. Creer un nouveau projet Hytale Model

Si vous voyez l'option Hytale Model, le plugin est installe avec succes.

## Formats supportes

Le plugin Hytale permet :

| Format | Utilisation |
|--------|---------|
| `.blockymodel` | Fichiers de modeles 3D |
| `.blockyanim` | Fichiers d'animation |
